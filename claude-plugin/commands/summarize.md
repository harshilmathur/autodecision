# /autodecision:summarize

Compress any Decision Brief into a one-page summary for sharing with teams, boards,
or stakeholders who won't read 300 lines.

## Invocation

```
/autodecision:summarize {slug}
/autodecision:summarize pricing-cut-20pct-full
/autodecision:summarize market-expansion-full
```

## Execution

1. Read `~/.autodecision/runs/{slug}/DECISION-BRIEF.md`
2. If file doesn't exist, list available runs and ask user to pick one.

3. Extract and compress into this format:

```markdown
# One-Page Summary: {decision statement}

**Recommendation:** {action — one sentence, no hedging}
**Confidence:** {level} | **Tilt:** {tilt} | **Converged:** {yes/no}

---

## Why This Is the Right Move
- {strongest argument 1 — grounded in data}
- {strongest argument 2}
- {strongest argument 3}

## Top Risks
- {risk 1 — probability, what triggers it}
- {risk 2}
- {risk 3}

## What Must Be True (load-bearing assumptions)

| Assumption | State | If Wrong |
|-----------|-------|----------|
| {assumption 1} | {SOLID/SHAKEABLE/FRAGILE} | {what happens} |
| {assumption 2} | | |
| {assumption 3} | | |

## Action Plan

| Phase | Timeline | Key Action | Kill Criteria |
|-------|----------|-----------|---------------|
| 1 | {when} | {what} | {stop if} |
| 2 | {when} | {what} | {stop if} |
| 3 | {when} | {what} | {stop if} |

## Watch For
- {monitoring signal 1 — frequency, trigger}
- {monitoring signal 2}
- {monitoring signal 3}

---

*Full analysis ({N} pages): ~/.autodecision/runs/{slug}/DECISION-BRIEF.md*
*Generated by Auto-Decision Engine | {date}*
```

4. Write to `~/.autodecision/runs/{slug}/SUMMARY.md`
5. Print to conversation
6. Offer publish or export:

   > "Share this summary?"
   > Options:
   > A) Publish — run `/autodecision:publish` (PDF → Notion, email, gist, Slack, Drive, or local)
   > B) Copy summary to current directory
   > C) Copy full brief to current directory
   > D) Skip

   If A: invoke the publish skill with the current slug.
   If B: `cp ~/.autodecision/runs/{slug}/SUMMARY.md ./{slug}-SUMMARY.md`
   If C: `cp ~/.autodecision/runs/{slug}/DECISION-BRIEF.md ./{slug}-DECISION-BRIEF.md`

## Rules

- Follow ALL human-readable output rules from decide.md (no snake_case, no raw IDs)
- The summary must be self-contained — a reader should understand the recommendation
  without ever reading the full brief
- Maximum ~40 lines of content (excluding table formatting)
- Every bullet is specific and grounded — no "various factors" or "multiple considerations"
